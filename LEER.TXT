requerimintos.txt


## Configurar entorno

1. Instalar pip3
```console
snap install pip3
```
2. Descargar virtualenv
```console
pip3 install venv
```

3. Creamos un direcctorio para trabajar y nos pocisionamos dentro.
```console
mkdir sopa_de_letras && cd sopa_de_letras

```
4. Crear un entorno local para el desarrollo
```console
python3 -m venv nombre_enviroment
```

5. Una ves que generamos nuestro entorno local, procedemos a activarlo e instalar las dependencias del proyecto.
```console
source nombre_enviroment/bin/activate  # Activar
pip install -r requerimientos.txt
```

6. Teniendo todos los requerimientos intalados
```console
python3 main.py
```
